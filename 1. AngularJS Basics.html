What is a scope and program with $scope and $rootScope? Scope is a JavaScript object that refers to the application model. It is an execution context for expressions. Scopes are arranged in hierarchical structure which mimic the DOM structure of the application. $scope is a special object created by AngularJS that contains the application model. It is available for use in Controllers, Directives and Expressions. $rootScope is the $scope created on the DOM element where the application is bootstrapped. It is the top-most scope and it is available for use in all parts of the application.
